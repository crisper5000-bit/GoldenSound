# Папки для медиафайлов

Для ручной загрузки файлов используйте эти директории:

- `backend/uploads/avatars` — аватарки пользователей
- `backend/uploads/covers` — обложки треков (jpg/png/webp)
- `backend/uploads/tracks` — mp3-файлы треков

Если указываете путь в БД вручную, используйте формат:

- `/uploads/avatars/<filename>`
- `/uploads/covers/<filename>`
- `/uploads/tracks/<filename>`
